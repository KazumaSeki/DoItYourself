# DoItYourself(未完成)

#### アプリの説明 
DIYで作成した作品を写真投稿するアプリです。
- 写真付きの投稿機能(投稿、編集、削除)
- 投稿に対してコメントができる機能(アイコン、ユーザー名、投稿内容、投稿日時が表示される)

#### Ruby,Railsのバージョン
- Ruby 3.0.0
- rails 6.0.3.6

#### DB
